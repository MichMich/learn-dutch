# Irregular Verbs

A reference of common irregular verbs

## Zijn (to be)

| Dutch | English |
| ------- | ----- |
| Ik **ben** | I am |
| Jij **bent** | You are |
| Hij/Zij/Het **is** | He/She/It is |
| Wij **zijn** | We are |
| Jullie **zijn** | You (*pl.*) are |
| Zij **zijn** | They are |
