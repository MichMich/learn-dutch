# Verbs

A reference of common verbs

## Komen (to come)

| Dutch | English |
| ------- | ----- |
| Ik **kom** | I come |
| Jij **komt** | You come |
| Hij/Zij/Het **komt** | He/She/It comes |
| Wij **komen** | We come |
| Jullie **komen** | You (*pl.*) come |
| Zij **komen** | They come |

## Willen (to want)

| Dutch | English |
| ------- | ----- |
| Ik **wil** | I want |
| Jij **wilt** | You want |
| Hij/Zij/Het **wil** | He/She/It wants |
| Wij **willen** | We want |
| Jullie **willen** | You (*pl.*) want |
| Zij **willen** | They want |
